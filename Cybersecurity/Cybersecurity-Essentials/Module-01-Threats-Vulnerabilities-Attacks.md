# Module 1: Cybersecurity Threats, Vulnerabilities, and Attacks

## Threat Domains

### Threat Categories
- Software attacks
- Human error
- Sabotage
- Theft
- Hardware failure
- Utility interruption
- Natural disasters

### Threat Sources
- Internal threats (employees, contractors)
- External attackers
- Social engineering

### Important Threats
- APT (Advanced Persistent Threat)
- Backdoor
- Rootkit
- Dark Web
- IOC (Indicator of Compromise)
- STIX & TAXII (Threat intelligence sharing standards)

---

## Deception (Social Engineering)

### Common Techniques
- Phishing
- Pretexting
- Quid Pro Quo
- Identity Theft
- Shoulder Surfing
- Dumpster Diving
- Piggybacking / Tailgating
- Watering Hole Attack
- Typosquatting

**Prevention**
- Security awareness training
- Verify identities
- Strong security policies

---

## Cyber Attacks

### Malware Types
- Virus
- Worm
- Trojan
- Ransomware
- Logic Bomb

### Network Attacks
- DoS
- DDoS
- DNS Spoofing
- DNS Hijacking

### Layer 2 Attacks
- ARP Spoofing
- IP Spoofing
- MAC Flooding
- Man-in-the-Middle (MITM)

### Other Attacks
- Zero-Day Attack
- Keylogger

**Defense**
- Firewalls
- Regular patching
- Load balancing
- Block unnecessary ICMP traffic

---

## Wireless & Mobile Attacks

### Common Attacks
- Grayware
- SMiShing
- Rogue Access Point
- Evil Twin
- Radio Jamming
- Bluejacking
- Bluesnarfing

### Wireless Security
- WEP
- WPA
- WPA2

**Best Practices**
- Change default passwords
- Detect rogue APs
- Use VPN
- Restrict guest access

---

## Application & Other Attacks

### Web Attacks
- XSS
- SQL Injection
- XML Injection
- LDAP Injection
- Buffer Overflow
- Remote Code Execution (RCE)
- CSRF
- Directory Traversal
- API Attacks

### Email Attacks
- Phishing
- Spear Phishing
- Whaling
- Vishing
- Pharming

### Other Threats
- Supply Chain Attack
- Cloud Attack
- Adversarial AI Attack

**Best Practices**
- Validate all user input
- Keep software updated
- Scan email attachments
- Use antivirus
